Canvas Libraries

CreateJS

A suite of Javascript libraries & tools for building rich, interactive experiences with HTML5.
  • EaselJS, TweenJS, SoundJS, PreloadJS
  • EaselJS is the one that actually helps drawing on Canvas

var canvas, stage; 
canvas = document.querySelector("#canvas");
stage = new Stage(canvas);
var g = new Graphics();
g.setStrokeStyle(1);
g.beginStroke(Graphics.getRGB(0,0,0));
g.beginFill(Graphics.getRGB(255,0,0));
g.drawCircle(0,0,20);
var s = new Shape(g);
s.x = 100;
s.y = 100;
stage.addChild(s);
stage.update();

KineticJS

KineticJS is an HTML5 Canvas JavaScript library that extends the 2d context by enabling high performance path detection and pixel detection for desktop and mobile applications. 

var stage = new Kinetic.Stage({
	container: "container",
	width: 500,
	height: 500
});
var layer = new Kinetic.Layer();
var rect = new Kinetic.Rect({
	x: 100,
	y: 100,
	width: 100,
	height: 100,
	fill: "#442d56",
	stroke: "black",
	strokeWidth: 2
});
layer.add(rect);
stage.add(layer);

FabricJS

Fabric.js is a framework that makes it easy to work with HTML5 canvas element. It is an interactive object model on top of canvas element. It is also an SVG-to-canvas (and canvas-to-SVG) parser.
  • Converts easily between SVG and Canvas code via JSON.
  • Allows real time manipulation of objects on the canvas.
  • Can serialize the whole canvas to JSON string!
  • Uses Node to build.

var canvas = new fabric.Canvas('canvas');
var dot = new fabric.Circle({
	left: 100,
	top: 100,
	radius: 20,
	fill: "#ffcc66",
	stroke: "#666666",
	strokeWidth: 3
});
canvas.add(dot);

CAKE

CAKE is a JavaScript scene graph library for the HTML5 canvas tag. You could think of it as SVG sans the XML and not be too far off.
  • No documentation! :(
  • Code base hasn't been updated in over a year. Dead?

var canvas = new Canvas(document.body, 500, 500);
var circle = new Circle(50, {
	id: 'theCircle',
	x: 100,
	y: 200,
	fill: "rgb(200,25,40)",
	stroke: "rgb(0,0,0)",
	strokeWidth: 3,
	endAngle: Math.PI*2
});
canvas.append(circle);

Boxbox

BoxBox is a wrapper for Box2DWeb. Box2dWeb is a physics engine. It is a port of Box2DFlash 2.1a to JavaScript.
  • Not a drawing lib for Canvas necessarily--just a wrapper for Box2DWeb, which is a port of Box2DFlash, which is a port of Box2D.

var scaleFactor = 30, // Convert Box2D's units to pixels
	canvas = document.getElementById("canvas"),
	world = boxbox.createWorld(canvas),
	ground = world.createEntity({
		color: "rgba(0,0,0,1)",
		x: 250 / scaleFactor,
		y: 480 / scaleFactor,
		width: 500 / scaleFactor,
		height: 10 / scaleFactor,
		type: "static"
	});

// Make a bunch of boxes
for (var i = 0, len = 10; i < len; i++) {

			y: Math.random() * -10,

			height: dim
		});

HTML5 Animation Tools

Adobe Edge Animate

Adobe® Edge Animate is a powerful, intuitive tool for creating stunning animated and interactive content using HTML5, CSS3 and JavaScript.
  • Intuitive user interface
  • Precision animation with the Timeline editor
  • Symbols/nested animations

Sencha Animator

Create Mobile CSS3 Animations with Ease.
  • Standards-based CSS3 animation
  • Output is clean and human readable.
  • Compatible with any mobile ad network.

Tumult Hype

Using Tumult Hype, you can create beautiful HTML5 web content. Animations and interactive content made with Tumult Hype works on desktops, smartphones and iPads. No coding required.
  • Does not use Canvas.
  • Produces valid, state-of-the-art HTML5, CSS3, and Javascript.
